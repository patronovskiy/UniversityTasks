Самая  простая,  но  неэффективная  (медленная)  реализация чтения/записи  данных  файла
может  быть  продемонстрирована  с использованием классов 
Reader – FileReader, Writer – FileWriter.
Основные методы:
–read() длячтения;
–write() для записи с удалением ранее имеющейся информации в файле;
–append() для добавления данных в файл (существующая ранее информация в файле не уничтожается).Самая  простая,  но  неэффективная  (медленная)  реализация чтения/записи  данных  файла
